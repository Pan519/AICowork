# AICowork 快速参考卡

> 快速上手指南

---

## 🚀 三步开始

```
1️⃣ 双击启动 AICowork
2️⃣ 设置 → Claude API → 输入 API Key → 保存
3️⃣ 开始对话！
```

---

## 📋 常用操作

### 对话操作

| 操作 | 方法 |
|------|------|
| 发送消息 | 输入框输入 → Enter |
| 换行 | Shift + Enter |
| 新建对话 | 点击 `+ 新对话` |
| 查看历史 | 侧边栏点击对话 |

### 斜杠命令

```
/help           - 显示帮助
/clear          - 清空对话
/memory save    - 保存到记忆
/memory search  - 搜索记忆
/memory ask     - 记忆问答
```

---

## 💾 记忆系统速查

### 添加记忆

1. 设置 → 记忆系统
2. 点击 `+ 添加新记忆`
3. 填写：标题、内容、分类、标签
4. 保存

### 创建标签

1. 设置 → 记忆系统 → 标签管理
2. 点击 `+ 新建标签`
3. 输入名称 → 添加

### 搜索记忆

搜索框输入关键词 → Enter

### 记忆分类

```
📁 项目相关      - 项目配置、目录结构
⚙️ 用户偏好      - 个人习惯、设置
💡 技术记录      - 代码片段、技术要点
📝 上下文信息    - 背景、上下文
💬 对话摘要      - 重要对话总结
🏷️ 自定义        - 自定义分类
```

## ⚙️ 常用设置

| 设置 | 位置 | 说明 |
|------|------|------|
| API Key | 设置 → Claude API | 必须配置才能使用 |
| 模型选择 | 设置 → Claude API | 选择 Claude 模型 |
| 启用记忆 | 设置 → 记忆系统 | 开启/关闭记忆功能 |
| 自动存储 | 设置 → 记忆系统 | 自动保存对话 |

---

## 🔧 故障排除

| 问题 | 解决方法 |
|------|----------|
| 无法发送消息 | 检查 API Key 是否配置 |
| 记忆搜索无结果 | 确认已添加记忆内容 |
| 应用启动慢 | 首次加载索引，正常现象 |
| 文件锁定错误 | 重启应用 |
