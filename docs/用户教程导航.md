### 给新用户的建议

```
1. 不要一次读完所有文档
   → 先看快速入门指南，边用边学

2. 多动手实践
   → 每学一个功能就立即试用

3. 善用搜索和查找
   → 遇到问题先在文档中搜索关键词

   → 将快速参考卡加入收藏夹
```
